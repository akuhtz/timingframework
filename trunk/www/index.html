Timing is Everything

Welcome to the TimingFramework project. This project contains source code for the "Timing is Everything" article, published here.

The main things to check out here are the Article, the source code, the Javadocs for the timing framework, and the SimpleAnimation sample application.

If you want to download and build the project (please do!), grab the appropriate files and either run "ant" in the main directory (where build.xml resides) or load it into NetBeans 4 (note the handy-dandy NetBeans project file in the "nbproject" directory).
